唐僧叨叨软件-涉及收集用户信息的第三方SDK说明

更新时间：2021年7月14日

生效时间：2021年7月14日

为保障唐僧叨叨软件的稳定运行或实现相关功能，我们可能会接入由第三方提供的软件开发包（SDK）。我们会对合作方获取个人信息的SDK进行严格的安全监测，以保护您的个人信息安全。我们将对接入的相关第三方SDK在以下目录中列明。您可以通过相关链接查看第三方的数据使用和保护规则。请注意，第三方SDK可能由于版本升级、策略调整等原因导致其个人信息处理类型发生变化，请以其公示的官方说明为准。

 

一、小米PUSH

使用目的：用于在小米设备上推送消息

收集个人信息类型：设备标识信息、设备位置信息、应用使用信息

官网链接：https://dev.mi.com/console/doc/detail?pId=1822

二、 华为PUSH

使用目的：用于在华为设备上推送消息

收集个人信息类型：应用标识信息、消息下发记录等

官网链接：https://developer.huawei.com/consumer/cn/doc/development/HMSCore-Guides/privacy-statement-0000001050042021

三、 VIVO PUSH

使用目的：用于在VIVO设备上推送消息

收集个人信息类型：设备标识信息、网络状态信息

官网链接：https://dev.vivo.com.cn/documentCenter/doc/366

四、OPPO PUSH

使用目的：用于在OPPO设备上推送消息

收集个人信息类型：设备标识信息、网络信息、设备状态信息、应用使用信息

官网链接：https://open.oppomobile.com/wiki/doc#id=10445

五、魅族 PUSH

使用目的：用于在魅族设备上推送消息

收集个人信息类型：设备标识信息、位置信息、网络状态信息、运营商信息

官网链接：https://open.flyme.cn/open-web/views/push.html

六、极光 PUSH

使用目的：用于无法设备的设备上推送消息

收集个人信息类型：设备标识信息、位置信息、网络状态信息、运营商信息

官网链接：https://www.jiguang.cn/license/privacy

七、Google FCM SDK

使用目的：用于海外安卓设设备的推送消息

收集个人信息类型：设备标识信息、位置信息、网络状态信息、运营商信息

官网链接：https://policies.google.com/privacy

八、高德地图

使用目的：地图及定位相关服务

收集个人信息类型：设备标识信息、位置信息

官网链接：https://lbs.amap.com/pages/privacy/

九、Bugly SDK

使用目的：监控TPNS自身SDK质量

收集个人信息类型：crash时包含本身SDK的堆栈信息

官网链接：https://privacy.qq.com/document/priview/fbd2c3f898df4c1c869925dd49d57827

十、X5内核（腾讯浏览服务）

使用目的：浏览网页数据，提高用户体验

收集个人信息类型：网络状态，读写SD卡，获取手机状态信息

官网链接：https://x5.tencent.com/tbs/guide/develop.html#5
